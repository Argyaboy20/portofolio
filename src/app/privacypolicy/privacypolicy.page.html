Privacy Policy

TLDR: rasfarras.my.id do not use cookies and we do not collect any personal data.

Website visitors

  • No personal information is collected.
  • No information is stored in the browser.
  • No information is shared with, sent to or sold to third-parties.
  • No information is shared with advertising companies.
  • No information is mined and harvested for personal and behavioral trends.
  • No information is monetized.

Information we collect and what we use it for

We run Vercel Analytics analytics on rasfarras.my.id. The following information is collected:

Page URL

We track the page URL of each page view on this website. We use this to understand which pages have been viewed and how many times a particular page has been viewed. For example: /privacy-policy, /about, /portfolio

HTTP Referrer

We use the referrer string to understand the number of visitors referred to this website from links on other sites. For example: https://vercel.com

Browser Information

We use this to understand what browsers people use when visiting this website. This is derived from the User-Agent HTTP header. The full User-Agent is discarded. For example: Chrome, Firefox, Safari

Operating System

We use this to understand what operating systems people use when visiting this website. We only collect the brand of the operating system. For example: macOS, Windows, Linux

Device Type

We use this to understand what devices people use when visiting this website. For example: Desktop, Mobile, Tablet

Country

We look up the visitor's country using their IP address. We do not track anything more granular than the country of origin and the IP address of the visitor is discarded. We never store IP addresses in our database or logs. For example: Indonesia, United States, United Kingdom

If you have any questions or concerns about this privacy policy, please contact me at: maulanafarras030520@gmail.com
